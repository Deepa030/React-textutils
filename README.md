# Text-utils React.js Project

Text-utils is a simple React.js project that provides a user interface for manipulating text. Users can convert text to uppercase, lowercase, and remove extra spaces with ease.

## Features

- **Convert to Uppercase:** Easily transform your text to uppercase.
- **Convert to Lowercase:** Change your text to lowercase effortlessly.
- **Remove Extra Spaces:** Clean up your text by removing extra spaces between words.
